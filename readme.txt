-------------- PARTS 1 AND 2 ------------------
-----------------------------------------------

To run our program, type the following into your terminal:
    
        python p4.py

To change the part that will be run when executing the above,
there is a variable in p4.py named

        TEST_PART

To test part 1, assign this variable to 1:

        TEST_PART = 1

To test part 2, assign this variable to 2:
        
        TEST_PART = 2
